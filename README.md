# Recipeek
Store and access a database of recipes for food to cook in a variety of formats.

## Dependencies
- POSIX-compliant system (sh)
- pandoc (For document conversion)
- fzf

## Example Layout of Recipe
```md
Servings
: Number of servings

# Hardware
- Stuff used to make the food

# Ingredients
- Food
```
